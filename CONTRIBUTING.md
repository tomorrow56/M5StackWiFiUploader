# Contributing to M5Stack WiFi Uploader

We welcome contributions to this project! Please follow these guidelines to ensure a smooth process.

## How to Contribute

1.  **Fork the repository** on GitHub.
2.  **Create a new branch** for your feature or bug fix.
3.  **Write your code** and add tests for your changes.
4.  **Ensure all tests pass**.
5.  **Submit a pull request** with a clear description of your changes.

## Reporting Bugs

- Use the **Bug Report** template on GitHub Issues.
- Provide a clear and concise description of the bug.
- Include steps to reproduce the issue, code snippets, and serial monitor logs.

## Feature Requests

- Use the **Feature Request** template on GitHub Issues.
- Explain the problem you are trying to solve and your proposed solution.

## Coding Style

- Follow the existing code style.
- Add comments to your code where necessary.
- Keep your code clean and readable.
